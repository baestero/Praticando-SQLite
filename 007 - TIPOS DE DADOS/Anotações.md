# INTEGER E TEXT

- INTEGER
  - Armazena números inteiros (boolean (0,1), timestamp)
- TEXT
  - Armazena textos (strings, datas, json)

# DADOS no SQLite

- Existem 3 momentos importantes em que o SQLite lida com tipo de dados:

  1.Definição de Tabelas
  - Definir o tipo de dado da coluna.

    2.Inserção de Dados

  - Converte o dado para o tipo de acordo com a finadade da coluna.

    3.Consulta de Dados

  - Retorna o valor armazenado.

# Tipos de Dados

- TEXT: Texto
- INTEGER: Números Inteiros
- REAL: Números flutuantes
- BLOB: Dados binários
- NULL: Valor nulo
