# UPDATE

## UPDATE cursos SET nome = 'HTML e CSS', aulas = 20 WHERE ID = 1

- Atualiza a coluna nome da tabela cursos para HTML e css E atualiza a coluna aulas para 20 onde o ID seja igual a 1.
  - O uso do WHERE é necessário, se não o script atualizará todos os registros da coluna nome e aulas para HTML e CSS e Aulas para 20.

## NÃO UTILIZAR UPDATE E DELETE sem WHERE
