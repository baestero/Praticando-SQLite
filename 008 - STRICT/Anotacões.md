# STRICT

- No modo strict, o SQLite impõe regras mais rígidas sobre os tipos de dados.

## CREATE TABLE

- Apenas alguns tipos de dados são permitidos.

## INSERT

- Os dados devem cosrresponder ao tipo definido na tabela

# Tipos de dados

1. INT\*: Números inteiros
2. INTEGER: Números inteiros
3. TEXT: TEXTO
4. REAL: Números flutuantes
5. BLOB: Dados Binários
6. ANY: Qualquer tipo de dado
