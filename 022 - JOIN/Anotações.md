# JOIN

- Agrega valores de duas tabelas com base em uma condição.
- O padrão do JOIN é o INNER JOIN, que retorna apenas linhas que têm corresponência em ambas tabelas. Pode usar JOIN, INNER JOIN ou apenas a virgula,.

# Múltiplos JOINs

- Podemos fazer múltiplos JOINs para agregar dados de várias tabelas. Se o nome da coluna for o mesmo em ambas tabelas, é necessário usar um alias AS.
